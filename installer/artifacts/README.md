# Installer Artifacts Drop Folder

Place final end-user installer binaries here so non-developers can just run setup:

- `windows/RoadGISProSetup.exe`
- `windows/RoadGISProSetup.msi`
- `linux/roadgispro_1.0.0_amd64.deb`
- `macos/sonoma/RoadGISPro.pkg`
- `macos/sequoia/RoadGISPro.pkg`
- `macos/tahoe/RoadGISPro.pkg`

Then users can launch:

- `installer/Run-Setup.bat` (Windows)
- `installer/run-setup.sh` (Linux/macOS)

and pick their OS/package in the UI.
